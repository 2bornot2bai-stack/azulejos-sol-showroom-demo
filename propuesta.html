Carta de presentación

Hola Lorenzo,

Te paso una primera demo visual que hemos preparado para que veas cómo podría evolucionar la web de Azulejos Sol.

La idea no es simplemente cambiar la web actual por una más moderna, sino convertirla en un showroom digital: una web más visual, a pantalla completa, adaptada a móvil y pensada para que el cliente pueda inspirarse, ver categorías, entender mejor lo que ofrecéis y pedir presupuesto de forma sencilla.

La demo es conceptual y usa contenido simulado, pero sirve para ver claramente la línea que proponemos: una presencia digital más profesional, más actual y mucho más alineada con la experiencia, producto y trayectoria que tiene Azulejos Sol.

Lo planteamos de forma escalonada: primero una web inicial de impacto, muy ajustada de precio, y después pequeñas mejoras opcionales si realmente os resultan útiles.
