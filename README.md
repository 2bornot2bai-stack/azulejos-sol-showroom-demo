# Demo showroom digital premium · Azulejos Sol

Demo estática creada como prototipo comercial para presentar una nueva web full-width, responsive y orientada a captación.

## Qué incluye

- `index.html`: estructura de la demo.
- `styles.css`: diseño premium responsive, grid full-width, microanimaciones y adaptación móvil.
- `app.js`: menú móvil, filtros de catálogo, formulario simulado, localStorage y transiciones.
- `assets/azulejos-sol-mark.svg`: marca visual ficticia para demo.
- `assets/favicon.svg`: favicon.
- `.nojekyll`: recomendado para GitHub Pages.

## Importante

Esta demo utiliza datos simulados e imágenes de referencia. No debe presentarse como versión final ni como sistema real de gestión.

En una versión real habría que sustituir:

- textos definitivos,
- imágenes propias de exposición/productos,
- datos de contacto reales,
- aviso legal, privacidad y cookies,
- marcas y catálogos autorizados,
- integraciones de formulario,
- analítica,
- CRM o panel interno si procede.

## Cómo publicarlo en GitHub Pages

1. Crear un repositorio nuevo.
2. Subir todos los archivos de esta carpeta a la raíz del repositorio.
3. Ir a Settings → Pages.
4. Seleccionar Branch: `main` y carpeta `/root`.
5. Guardar.
6. GitHub generará una URL pública de demo.
